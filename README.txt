Bonjour,

Afin de réaliser ce projet, je me suis basé sur le simple thème par défaut de Wordpress : Twenty Twenty-One, j'ai donc fait un thème enfant de ce thème.
J'aurais pu créer un thème en commençant de zéro mais je pense que je n'aurai pas eu le temps de tout intégrer. 

J'ai donc utilisé le nouvel éditeur de Wordpress : Gutenberg.
Ce nouvel éditeur nous permet de créer des blocs réutilisables sur les pages que l'on crée. 
J'ai donc créé un ensemble de blocs qui s'appelle Audit UX, celui-ci regroupe tous les blocs de la page Audit UX.

Si j'avais eu du budget pour monter ce site toujours sans page builder, j'aurai choisi d'utiliser le plugin Advanced Custom Fields qui nous permet de créer des blocs modulaires avec les "Flexible Content fields".

Je n'ai pas réussi à intégrer plusieurs choses dans la page : 
- Le bloc "Combien coûte un audit de l'expérience utilisateur" avec son background transparant
- Ainsi que l'interface exacte du sous-niveau du menu mobile

Pour m'aider dans ma tâche j'ai utilisé plusieurs plugins :
- "Options for Twenty Twenty-One" qui rajoute quelques options pour le thème Twenty Twenty-One
- "Max Mega Menu" que j'ai utilisé pour faire le menu 
- "Contact Form 7" que j'ai utilisé pour le formulaire de contact
- "Breadcrumb NavXT" pour rajouter les breadcrumbs sur les pages
